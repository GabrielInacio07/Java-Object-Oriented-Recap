# ☕ Java – Recapitulação de Programação Orientada a Objetos (POO) (Módulo 09)

Este repositório contém **exercícios e exemplos de recapitulação** relacionados à **Programação Orientada a Objetos (POO)** em Java, desenvolvidos ao longo do **Módulo 09 do curso da plataforma Alura**.

O foco deste módulo é **fixar os principais conceitos de orientação a objetos** por meio de prática e revisão de código.

---

## 📚 Conteúdos revisados

Este módulo contém exercícios e exemplos que reforçam os seguintes conceitos de POO:

- 📦 **Classes e Objetos**  
- 🔐 **Encapsulamento**  
- 🧠 **Métodos e Comportamentos**  
- 💡 **Instanciação de objetos**  
- 🧩 Aplicação prática dos princípios de POO no código

Estes conceitos são revisitados com exemplos simples executáveis no método `main()` para reforçar seu uso no Java.

---
*(A estrutura pode variar conforme a organização do projeto. Ajuste conforme necessário.)*

---

## 🎯 Objetivo deste módulo

Este repositório faz parte da minha **jornada de estudos em Java**, com os objetivos de:

- Revisar e consolidar os conceitos de Programação Orientada a Objetos  
- Praticar a criação de classes e objetos e seu uso  
- Demonstrar lógica e sintaxe de POO em exemplos concretos  
- Preparar para conteúdos mais avançados após POO
